This course encompasses topics such as HTML, CSS, an introduction to JavaScript, and the creation of websites.

Prerequisites: Successful completion of MAT 100 or an equivalent math placement score. Additionally, a fundamental understanding of computer operations is required, including skills such as copy and paste functions, proficiency in using a text editor, navigating a web browser, familiarity with folder management, uploading and downloading files, and the ability to zip and unzip files. Some students may also need to install supplementary software, such as JetBrains' WebStorm for HTML file development and an image-editing software like Paint.NET.

Description: This course adopts a problem-solving oriented methodology to acquaint students with the diverse facets of website production. Topics covered include requirements elicitation and analysis, design specifications, handling of images, videos, audio, HTML, CSS, and the application of mathematical/logical programming through a scripting language like JavaScript. Engaging in interactive, hands-on sessions, students will have the opportunity to build a website that fulfills a predefined set of requirements. The successful completion of this course meets General Education requirements for Computer Science and Quantitative Reasoning.

Objectives:
• Apply a standard problem-solving approach to website design and implementation.
• Discuss the fundamental components of the website production process, including requirements elicitation and analysis, design specifications, implementation using HTML, CSS, and JavaScript, as well as outcome evaluation.
• Demonstrate the ability to construct a moderately complex website featuring text, images, audio, video, and JavaScript functions.

Quantitative Reasoning Learning Outcomes:
• Make informed judgments and draw conclusions about quantitative information, recognizing its limitations.
• Apply a stepwise problem-solving methodology to analyze quantitative problems, identifying knowns, unknowns, and requirements.
• Use quantitative principles, theories, and methods to solve real-world problems.
• Solve numerical problems by creating appropriate sets of equations and translating them into algorithmic solutions, including error checking, algebraic computations, numerical data analysis, and 2D graphics.
• Apply computational procedures to solve problems effectively.
• Utilize a scripting language, such as JavaScript, to design and implement functions based on algorithmic solutions to mathematical and logical problems for incorporation into webpages.
• Explain quantitative information presented in textual, visual, or notational forms.
• Produce various charts, graphs, tables, etc., based on algorithmic solutions to numerical problems.
• Present quantitative information in clear and appropriate forms.
• Integrate various charts, graphs, tables, etc., derived from algorithmic solutions to numerical problems into a website's design.
