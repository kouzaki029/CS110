To create a list in HTML, you can use one of the three tags, <ul> , <dl> , and <ol>.


<h3>Unordered List</h3>
<ul>
   <li>Item 1</li>
   <li>Item 2
      <ul>
         <li>Subitem 2.1</li>
         <li>Subitem 2.2</li>
      </ul>
   </li>
   <li>Item 3</li>
</ul>

<h3>Descriptive List</h3>
<dl>
   <dt>Term 1</dt>
   <dd>Definition 1</dd>
   <dt>Term 2</dt>
   <dd>Definition 2</dd>
   <dt>Term 3</dt>
   <dd>Definition 3</dd>
</dl>

<h3>Ordered List (Numbered)</h3>
<ol>
   <li>Step 1</li>
   <li>Step 2
      <ol>
         <li>Substep 2.1</li>
         <li>Substep 2.2</li>
      </ol>
   </li>
   <li>Step 3</li>
</ol>

Figure 2: Three Types of Lists in HTML

The initiation of an unordered list involves using the <ul> tag, and each individual list item begins with the <li> tag.

By default, the list items will be denoted with small black circles, representing bullets.

Commencing an ordered list requires the use of the <ol> tag, and each item within the list begins with the <li> tag.

By default, the list items are numbered.

HTML additionally accommodates description lists.

A description list entails a compilation of terms, each accompanied by a corresponding description.

The <dl> tag outlines the description list, the <dt> tag delineates the term (i.e., the name), and the <dd> tag furnishes the description for each term.
