The <script> tag in HTML serves to declare a client-side script, typically JavaScript. This element can either encapsulate script statements directly or reference an external script file via the src attribute.

JavaScript finds frequent application in tasks such as image manipulation, form validation, and making dynamic alterations to content.

For selecting HTML elements, JavaScript commonly employs the document.getElementById() method.

In the following JavaScript illustration, the script changes the text of an HTML element when a button is clicked.


<body>
   <h1 id="changeMe">
      Hello, there!
   </h1>

   <button onclick="changeText()">
      Click me!
   </button>

   <script>
      function changeText()
      {
         var element = document.getElementID("changeMe");

         element.textContent = "Hello, HTML Beginners!";
      }
   </script>
</body>
