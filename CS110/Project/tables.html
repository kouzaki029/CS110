Tables can be defined using the <table> tag, <tr> tag, <th> tag, and <td> tag.


<table>
   <tr>
      <th>Title</th>
      <th>Author</th>
      <th>Publication Year</th>
   </tr>
   <tr>
      <td>The Great Gatsby</td>
      <td>F. Scott Fitzgerald</td>
      <td>1925</td>
   </tr>
   <tr>
      <td>To Kill a Mockingbird</td>
      <td>Harper Lee</td>
      <td>1960</td>
   </tr>
   <tr>
      <td>1984</td>
      <td>George Orwell</td>
      <td>1949</td>
   </tr>
</table>

Figure 2: A Simple Table in HTML

To create a table, you need to first include the <table> tag, followed by an ending </table> tag to indicate where the last line of the table is.

Within the <table> tags, a following <tr> tag must be included to indicate that a new table row is being created.

Within the <tr> tags, data is added via the <th> tag, or the <td> tag.

The <th> tag, states that the data is a header (e.g., "Wild Life Animals", "Giraffes", "Elephants", "Lions", "Zebras", "Kruger Park", "Moremi Park", and "Chobe Park" are all example of a table header)

Whereas, the <td> tag, state the data is a normal piece of data (e.g., "5", "127", "23", "1", "1", "0", "0", "1", "1", "0", and "0" are all examples of a table data).

Similarly to the <table> tag, both the <th> tag and the <td> tag must have an ending tag. (i.e., </th> , </td> respectively).
