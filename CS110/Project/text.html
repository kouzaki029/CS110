Headings in HTML are specified using the <h1> to <h6> tags.

<h1> denotes the most significant heading, while <h6> designates the least significant heading. Primary headings, designated by <h1> , should be employed for main topics, followed by <h2> for subtopics, and subsequently, less prominent headings like <h3> , and so forth.

Paragraphs in HTML are indicated using the <p> tag.

Each paragraph invariably begins on a new line, and web browsers automatically include some white space (margin) both preceding and following a paragraph.

The presentation of HTML content is unpredictable. Varied screen sizes, whether large or small, and the resizing of windows yield diverse outcomes.

In HTML, modifying the display by inserting additional spaces or lines in the code is ineffective. Any surplus spaces and lines in the HTML code will be automatically eliminated by the browser upon page display.

HTML utilizes the <a> tag to define links. The destination of the link is indicated in the href attribute. Attributes are employed to furnish extra details regarding HTML elements.

HTML images are defined with the <img> tag.

The source file (src), alternative text (alt), width, and height are provided as attributes.

The <hr> tag establishes a thematic break within an HTML page, typically manifesting as a horizontal rule. Primarily, the <hr> element serves to segregate content or delineate a shift in an HTML page. The <hr> tag is considered an empty tag, signifying that it lacks a closing tag.

The <br> element in HTML designates a line break. Employ <br> when you wish to introduce a line break (a new line) without initiating a new paragraph. This tag is an empty tag, signifying the absence of an end tag.
